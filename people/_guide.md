# People profile guide

Each person lives in `people/<slug>/` with:

- `IDENTITY.md` — short “nameplate” (name, vibe, one-liner)
- `SOUL.md` — operating principles for an agent

## Recommended SOUL sections

- Core Truths
- Operating Principles
- Communication Style
- Boundaries
- Blind Spots / Failure Modes

## Living people

Be extra conservative:
- Avoid insulting language.
- Avoid diagnosing mental health.
- Avoid speculation about motives as facts.
- Prefer direct quotes with dates.
